# Design Skills for Claude Code

一套专业级 Claude Code 设计技能包，让你的 AI 助手变身为资深 UI/UX 设计师。

## 📦 包含技能

| 技能 | 描述 |
|------|------|
| [ui-designer](./ui-designer/) | 专业 UI/UX 设计师 — 自动市场调研 + 完整设计方案 + 代码实现 |

## 🚀 快速开始

### 安装

将此仓库克隆或下载到你的 Claude Code 技能目录：

```bash
# 方式一：克隆到 skills 目录
cd ~/.claude/skills
git clone https://github.com/<你的用户名>/design-skills.git

# 方式二：手动下载后复制
# 下载 zip → 解压 → 将 ui-designer 文件夹放到 ~/.claude/skills/ 下
```

### 使用

在 Claude Code 中直接描述你的设计需求，技能会自动激活：

```
# 示例对话
"帮我设计一个科技产品的落地页"
"设计一个 SaaS 产品的定价页面"
"我需要一个暗色风格的个人作品集网站"
```

Claude 会自动执行：
1. **市场调研** — 搜索 3-5 个同类型优秀设计作为参考
2. **设计方案** — 输出配色、字体、布局、动效的完整方案
3. **代码实现** — 生成可直接运行的 HTML + CSS + JS 代码

## 🎨 技能特色

- **自动市场调研**：先分析竞品设计，再动手设计，拒绝闭门造车
- **严格设计规范**：涵盖图标、排版、配色、布局、动效、响应式等 8 大规范体系
- **专业 SVG 图标**：禁止使用 emoji，全部使用矢量图标
- **多风格支持**：科技硬件、奢侈品牌、极简现代、暗黑未来、活力年轻 5 大风格模板
- **3D 模型集成**：内置 Google model-viewer 集成方案
- **代码即设计**：输出可直接运行的前端代码，所见即所得

## 📐 设计规范概要

### 排版层级

| 层级 | 用途 | 字号范围 | 字重 |
|------|------|----------|------|
| H1 | 页面主标题 | 42px - 80px | 700+ |
| H2 | 区块标题 | 28px - 52px | 700 |
| H3 | 卡片/子标题 | 17px - 24px | 600-700 |
| Body | 正文 | 14px - 18px | 400 |
| Caption | 标签/注释 | 11px - 13px | 500-600 |

### 配色变量示例

```css
:root {
  --primary: #76B900;       /* 主色 */
  --primary-dark: #5a8f00;  /* 悬停态 */
  --bg: #000000;            /* 背景色 */
  --surface: #111111;       /* 卡片面板 */
  --ink: #ffffff;           /* 主文字 */
  --ink-light: #a0a0a0;     /* 次要文字 */
}
```

### 断点设置

| 断点 | 宽度 | 设备 |
|------|------|------|
| 移动端 | < 768px | 手机 |
| 平板 | 768px - 1024px | 平板 |
| 桌面端 | > 1024px | 笔记本/台式机 |

## 💡 使用技巧

### 让设计更精准

1. **明确风格偏好**：告诉 Claude 你想要的风格（如"暗色科技风"、"极简白色"）
2. **提供参考案例**：如果有喜欢的网站，直接告诉 Claude 网址
3. **指定目标受众**：说明产品面向的用户群体，影响设计决策
4. **分区块迭代**：一次只设计一个区块，逐步完善

### 高级用法

```
"参考 apple.com 的设计语言，为我的耳机产品设计一个落地页"
"用暗黑未来风格设计一个区块链项目的首页，主色用霓虹蓝"
"帮我把现有页面改造成移动端友好的响应式设计"
```
